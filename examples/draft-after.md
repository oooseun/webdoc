# Pulse: a faster way to read your analytics

Pulse is an analytics dashboard for teams that have outgrown exported spreadsheets. It loads a year of event data in about two seconds and refreshes as new events land, so the figure you cite in a meeting matches the figure on the screen.

The dashboard ships with a live event stream, a funnel builder, and a retention grid. They share one query engine, so a filter set in any view carries to the rest. Connecting a data source takes a single API key and about a minute, with no schema to define first.

We built Pulse because our old tools made simple questions slow. A teammate who wants yesterday's signups by channel should be able to get them without filing a ticket. Pulse puts that query one click away and exports the answer as a CSV file or a shareable link.

Pulse is in open beta now. The free tier covers five million events a month, and the full pricing and retention policy are on the docs site.
